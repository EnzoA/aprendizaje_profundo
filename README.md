### Requerimientos
* Lenguaje de Programación
    * Python 3.8
    * Pip para instalar librerías
* Librerías
    * Numpy 1.18
    * SciPy 1.5
    * PyTorch
* Consola Interactiva de Python 
    * IPython
* Herramientas
    * PyTest para tests
    * GitHub para repositorios
* IDE Recomendado 
    * PyCharm Community Edition
